# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [2.0.0] — 2026-03-12

### Changed
- Refactored from a single HTML file into a maintainable web app structure:
  - `index.html` — markup only
  - `css/style.css` — all styles and design tokens
  - `js/main.js` — all JavaScript (navigation, modal, charts)
- Translated all code comments to English

---

## [1.1.0] — 2026-03-12

### Added
- Full inline code documentation (Italian) across CSS, HTML, and JavaScript

---

## [1.0.0] — 2026-03-12

### Added
- Single-file interactive Six Sigma reference tool (`bb-stats-reference.html`)
  - 9 tabs: Quick Reference, Capability, Control Charts, MSA/GRR,
    Hypothesis Tests, Regression, DOE, Sampling, Non-Parametric
  - 17 Chart.js charts
  - All statistical formulas with beverage industry worked examples
  - Responsive layout — dark theme — no build tools required
